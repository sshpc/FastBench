# FastBench跑分 Linux性能测试一键脚本 

* 快速测试 1分钟左右可结束
* 测试结果仅供参考

## 一键执行

```sh
wget -N  http://raw.githubusercontent.com/sshpc/FastBench/main/FastBench.sh && chmod +x FastBench.sh && sudo ./FastBench.sh
```
